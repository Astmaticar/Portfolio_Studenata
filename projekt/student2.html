Projekti

Webshop

Full-Stack web trgovina s modernim korisničkim sučeljem razvijenim u React-u. Backend implementiran pomoću Node.js i Express frameworka, s MySQL bazom podataka i Sequelize ORM-om. Projekt uključuje potpunu autentifikaciju korisnika, upravljanje proizvodima, košaricu te administracijsku nadzornu ploču za upravljanje poslovnom logikom.

Web-knjižnica

Full-Stack aplikacija za upravljanje knjižnicom razvijena u Django okviru. Frontend integriran s Django template sistemom omogućava korisnicima prijavu, pregled kataloga dostupnih knjiga, najam literarture te mogućnost rezervacije trenutno zauzetih knjiga. Backend osigurava upravljanje korisnicima, knjigama i transakcijama iznajmljivanja.

Django Python SQLite Full-Stack
Simulacija Digital Twin City

Full-Stack web sustav za simulaciju digitalnog blizanca grada. Frontend razvijen u React frameworku integrira Mapbox GL JS za interaktivnu kartografiju i vizualizaciju urbanih podataka. Backend baziran na Node.js/Express arhitekturi koristi Socket.io za real-time dvosmjernu komunikaciju između servera i klijenata, omogućavajući live praćenje promjena u gradskom sustavu.
