⚠️ Što Trebate Dodati

1. Slike (Obavezno - 4+ formata)

Trebate dodati slike u images/ folder:

JPG Format:

  • gallery-1.jpg (800x600px) - Veća rezolucija
  • gallery-1.jpg (300x225px) - Thumbnail

PNG Format:

  • gallery-2.png (800x600px)
  • gallery-2.png (300x225px)

GIF Format:

  • gallery-3.gif (800x600px)
  • gallery-3.gif (300x225px)

AVIF Format (Moderni):

  • gallery-4.avif (800x600px)
  • gallery-4.avif (300x225px)
💡 Gdje preuzeti slike:
• Unsplash.com (besplatno)
• Pixabay.com (besplatno)
• Pexels.com (besplatno)
• Convertio.co - za konverziju između formata

2. Audio Datoteke (Obavezno)

Trebate dodati audio u audio/ folder:
  • student1.mp3 - Audio predstavljanje Lana Gale (max 15 sec)
  • student2.mp3 - Audio predstavljanje Dorijan Dragić (max 15 sec)
💡 Kako kreirati audio:
• Audacity.org (besplatno, desktop)
• VoiceRecorder.com (online)
• OkruPodcast.com (za uređivanje)
• Trebate samo kratko predstavljanje sebe (15 sekundi)

3. Video Datoteke (Opciono - YouTube je već Embedded)

Video je već embedded sa YouTube linkova:
• student1.html: Osijek video
• student2.html: Našice video

Ako želite lokalne MP4 datoteke: trebate dodati u video/ folder

🚀 Kako Pokrenuti/Testirati Projekt

Metoda 1: VS Code Live Server (Preporučeno)

  1. Otvorite projekt u VS Code
  2. Instalirajte ekstenziju "Live Server" (Ritwick Dey)
  3. Desni klik na index.html → "Open with Live Server"
  4. Projekt će biti dostupan na http://localhost:5500

Metoda 2: Python Simple Server

  1. Otvorite terminal u projektu
  2. Pokrenite: python -m http.server 8000
  3. Pristupite na http://localhost:8000

Metoda 3: PHP Server

  1. Pokrenite: php -S localhost:8000
  2. Pristupite na http://localhost:8000
